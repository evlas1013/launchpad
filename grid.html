A* Mapping Algorithm

Using the grid above, set a Green Box (Starting Point) and a Red Box (End Point).

You may set as many obstacles as you would like.

Use the radio buttons below to pick which type of box you would like to set and click inside the grid to set it.

When you're done, click the "Map Green To Red" button to trace the shortest possible route. If not route is available, an alert will pop up after a few moments informing you.

Gray boxes show all possible routes mapped.

Be sure to click reset between maps. Failing to do so will cause the program to stop responding. I am working to resolve this.

Enjoy!
